Simple deploy script

  • Jul 26, 2020
  • One minute read

A script to automatically push to Github, then deploy to Netlify.

To display the code correctly be sure to click the “Read more” tag below.

cd /

cd your_directory

rm -rf public

hugo

git init

git add public

git commit -m ‘First commit’

git remote add origin https://github.com/github_username/repo_name.git

git fetch origin master

git push -f origin master

Notes:

  1. There are single quotes around ‘First commit’ Use double quotes only if you are using Windows.

  2. -f after push means force. Use with caution.
